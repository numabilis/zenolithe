First HTML Page

Under the PHP-INF/lib folder create a samples subfolder . All libraries and code (including your application code) stands in the PHP-INF/lib folder.

Within your samples folder add a views subfolders. You'll store your view's files in the folderr. We will

Within the samples/views folder create an html file called sample1.php :

<!doctype html>
<html>
    <head>
        <title>Zenolithe sample 1</title>
    </head>
    <body>
        First zenolithe HTML page !
    </body>
</html>

In your browser go to to http://localhost/myapp/sample1.php.

That's it!

The sample1.php file being a PHP file you can code as usual within the samples/views folder. BUT IT'S A REALLY BAD THING! So this step just show you how to use zenolithe without using it! Great... See now how to use zenolithe while creating your first controlled page.
