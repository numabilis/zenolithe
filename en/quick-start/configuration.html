Configuration

You configure your application using files in the PHP-INF/conf folder. We will see later in details the configuration files.

To add your own application, samples for example, create a folder samples under the PHP-INF/lib folder and add 'lib/samples' in the includepath.conf.php file which must look like:

return array(
    'lib/zenolithe/',
    'lib/samples/'
);

That's it! Your application or library folder samples is now referenced by zenolithe and can be processed.

You can now create your first HTML page for your application.
