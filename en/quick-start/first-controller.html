First Controller

In this installment, we'll see how to setup zenolithe to handle a request with our own controller. The exercice is to show a page that display the current current date and time. Amazing!

Define the route

As in the preceding example we define the route file sample3.php under the routes folder:

return array(
    'controller' => array(
        'class' => 'org\zenolithe\samples\ShowDateTimeController'
    )
);

This file declare a route (sample3.php) for which the request is handled by a controller which is an instance of the org\zenolithe\samples\ShowDateTimeController.

Define the controller

All the application's code is stored in a code folder under the PHP-INF/lib/samples folder. So we need to create it.

The controller class is defined using namespace org\zenolithe\samples, so we need to create all these subfolders under the code folder and the ShowDateTimeController.php file in it. So we have the file ShowDateTimeController.php located in PHP-INF/lib/samples/org/zenolithe/samples. This file define the controller class which should implement the org\zenolithe\kernel\mvc\controllers\IController interface which define only one method:

public function handleRequest(Request $request);

Our controller look like:

namespace org\zenolithe\samples;

use org\zenolithe\kernel\http\Request;
use org\zenolithe\kernel\mvc\SimpleModel;
use org\zenolithe\kernel\mvc\controllers\IController;

class ShowDateTimeController implements IController {
    public function handleRequest(Request $request) {
        $model = new SimpleModel();
        
        $model->set('datetime', time());
        
        $model->setViewName('sample3-datetime');
        
        return $model;
    }
}

It defines a model whith some data, datetime which contains the current datetime. and define the view which has to be used to render the response (sample3-datetime).

Define the view

We create the sample3-datetime.php view file under the views folder :

<!doctype html>
<html>
    <head>
        <title>Zenolithe sample 3</title>
    </head>
    <body>
        First controller! Executed on <?php echo date('F j, Y, g:i:s a', $model->get('datetime')); ?>
    </body>
</html>

In your browser go to to http://localhost/myapp/sample3.php.

That's it!

This is a basic example that could be extended in many ways.
