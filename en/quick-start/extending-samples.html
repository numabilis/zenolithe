Extending the samples

The samples in this quick start are very basics and there are many ways to extend them:

Basic mechanisms

  • Reusing a crontroller for several requests with different views
  • Reusing parts of views
  • Exploring the power of the dependancy injection mechanism
  • Using the interception mechanisme
  • Using the logging engine
  • Redirecting and forwarding the request

Standard web applications features

  • Adding database support to your application
  • Adding a form
  • Localizing the application
  • Using sessions
  • Access control

Extending the Zenolithe framework

  • Using third party libraries
  • Use your own localizing mechanism
  • Use your own routing mechanism
  • Use your own viewing mechanism
