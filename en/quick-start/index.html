Installation

Prerequisites

In order tu use Zenolithe, you need :

  • a web server up and running
  • a url rewriting engine (mod_rewrite for Apache)
  • PHP 5.5 or greater

Installation

  • Download the latest archive zip file or tar ball or download a release.
  • Unpack it.
  • Copy the the content of the app folder in a folder under the web document root.

That's it !

For this documentation we assume that you have installed the framework in a folder called myapp in the web server root document /var/www/. All the files are located in /var/www/myapp/

You now need to configure your application.
