First Controlled page

We refer as a controlled page a page that is served using a controller.

Define a route

First of all we need to tell what is the URL of that page. If we want to access this page using http://localhost/myapp/sample2.php for example we need to declare a route to sample2.php.

Create a routes subfolder under PHP-INF/lib/samples. This folder will store all the routes definitions. A route is defined by a php file which location inside the routes folder is the same as the URL part from the app folder (myapp) in the browser location bar. For example if you want to define a route for the URL http://localhost/myapp/path/to/the/file.php, just create the file : PHP-INF/lib/samples/routes/path/to/the/file.php.

For our example http://localhost/samples/sample2.php we create a file sample2.php inside the routes folder :

return array(
    'controller' => array(
        'class' => 'org\zenolithe\kernel\mvc\controllers\SimpleController',
        'parameters' => array(
            'viewName' => 'sample2-view'
        )
    )
);

This file declare a route (which is the file path under the routes folder) for which the request is handled by a controller which is an instance of the org\zenolithe\kernel\mvc\controllers\SimpleController class which has the viewName property setted to sample2-view. SimpleController is a controller defined by the zenolithe framework that render the response with the file defined by the viewName property. Be carrefull that the name doesn't have the .php extension.

Later you will learn to create your own controllers to handle requests.

Define a view

As for the frist sample, we define a sample2-view.php file under the views folder:

<!doctype html>
<html>
    <head>
        <title>Zenolithe sample 2</title>
    </head>
    <body>
        First zenolithe controlled page !
    </body>
</html>

In your browser go to to http://localhost/myapp/sample2.php.

That's it!

It's time for you to create your first controller.
